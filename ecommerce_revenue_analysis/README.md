# E-Commerce Revenue Analysis

This project analyzes an e-commerce dataset to discover revenue trends,
top performing products, and country level sales patterns.

## Dataset
Online Retail dataset containing 541,909 transactions.

## Tools Used
Python
Pandas
Matplotlib
Jupyter Notebook

## Key Analysis
• Data cleaning and preprocessing
• Top 10 products by revenue
• Revenue by country
• Sales distribution analysis
